# Telegram-бот поддержки

Бот отвечает в личном чате, показывает три кнопки (inline), принимает текст по выбранному сценарию и при необходимости пересылает сообщение оператору.

## Частые причины «бот молчит»

1. **Процесс не запущен** — в терминале должна быть строка `polling started. Bot: @...`.
2. **Другой webhook** — если бот когда-то подключали к серверу, обновления уходят туда, а не в твой `npm start`. Скрипт при старте снимает webhook; проверь: `node check-token.js` (должно быть `webhook URL: (пусто)`).
3. **Два разных бота** — ссылка в сайте открывает одного @username, а токен в `.env` от другого бота.
4. **Старая reply-клавиатура** — после `/start` бот сначала снимает клавиатуру снизу, затем показывает кнопки под сообщением; если не видишь кнопок, обнови чат или нажми `/menu` ещё раз.

## Создание бота с нуля

### 1. Токен в Telegram

1. Открой [@BotFather](https://t.me/BotFather).
2. Отправь `/newbot` (или `/mybots` → выбери бота → **API Token**).
3. Задай имя и username (например `purple_support_bot`).
4. Скопируй **токен** вида `123456789:AAH...` — он больше нигде не показывается полностью, сохрани его.

### 2. Проект на компьютере

```bash
cd telegram-bot
npm install
cp .env.example .env
```

Открой `telegram-bot/.env` и вставь:

- `TELEGRAM_BOT_TOKEN=` — токен от BotFather.
- `SUPPORT_OPERATOR_CHAT_ID=` — опционально: id чата, куда слать копии обращений (или оставь пустым для проверки).

### 3. Проверка токена

```bash
node check-token.js
```

Должно вывести `username`, `webhook URL: (пусто)`.

### 4. Запуск

```bash
npm start
```

В консоли: `webhook after delete: (none)` и `polling started. Bot: @...`.

### 5. Отладка в Telegram

1. Открой в Telegram чат с именем **`@username`** из лога (не с другим ботом).
2. Отправь `/start` — должны появиться три кнопки под сообщением «Выбери действие:».
3. Если непонятно, что приходит на сервер, в `.env` добавь `SUPPORT_BOT_DEBUG=1` и перезапусти `npm start` — в консоли будут строки вида `[telegram-bot] message: ...` и `callback_query: m:site`.

## Как узнать SUPPORT_OPERATOR_CHAT_ID

- Личный чат: напиши [@userinfobot](https://t.me/userinfobot), скопируй `Id`.
- Группа: добавь бота в группу, напиши в группу любое сообщение; временно залогируй `ctx.chat.id` в коде или используй бота, который показывает id группы.

## Команды

| Команда | Действие |
|--------|----------|
| `/start` | Приветствие, снять старую клавиатуру, показать меню |
| `/menu` | То же меню ещё раз |
